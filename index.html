Uvod u programski jezik „Python“


1. Sta je Python?


Programski jezik koristi se za pisanje (i koriscenje) programskog koda.

Python je jedan od najpopularnijih programskih jezika. Ovaj programski jezik spada u kategoriju interpretiranih programskih jezika.
Interpretirani programski jezik je jezik koji objasnjava drugom (masinskom) programskom kodu sta treba da radi. Python je interpretirani programski jezik, jer interpretira (objasnjava) napravljen kod drugom (mašinskom) programskom jeziku, koji ce biti sastavljen u binarni (racunarski) jezik.

Python se koristi dnevno u svakodnevnom zivotu. Vecina popularnih kompanija koriste Python za usluge koje pruzaju.

Primeri kompanija koje koriste Python su: Google (Youtube, Google Pretraživač, Gmail), Meta (Facebook, Instagram), Spotify, Netflix…


2. Instalacija Programskog Jezika Python


Python se moze preuzeti sa zvanicnog Python sajta (https://www.python.org/downloads/)

Instrukcije za preuzimanje Python-a:

  1. Uci na sajt https://www.python.org/downloads/
  2. Pritisnuti dugme “Download Python (verzija)”
  3. Nakon preuzimanja (i pokretanja “Python Setup” aplikacije), pritisnuti “Install Now”
  4. (BONUS) Pritisnuti “Add Python To Path”, za naprednije Python funkcije.

Nakon preuzimanja zatvoriti Setup aplikaciju.
